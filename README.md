

# Expense-Tracker

Simple expense tracker built with MERN stack

### How to run

 npm install
 
 cd client npm install
 cd ..
 
 ### Run front and backend
 npm run dev
 
 ### Backend only
 npm run server
 
 ### Frontend only
 npm run client
 
 ### Build client
 cd client
 npm run build
